# HR — 업무 PC 파일 정리 규칙

> **적용 환경** : 폐쇄망 · 사내 업무 포털(그룹웨어) · 사내 메신저·메일 파일 첨부 전달 · NAS / SVN 공유 드라이브

> ⚠️ **HR 폴더는 HR 담당자 전용 NAS 접근 권한으로 운영합니다.**

---

## 핵심 원칙

| 원칙 | 내용 |
|------|------|
| 숫자 접두사 정렬 | `00_` → `01_` → `20_` 순으로 탐색기 정렬 고정 |
| 업무보고 최우선 | `00_업무보고`가 항상 최상단 노출 |
| 개인정보 메신저 금지 | 개인정보 파일은 메신저 전달 금지 — 사내 메일(암호화)만 허용 |
| 수신함 당일 정리 | 메일 수신 입사서류는 당일 암호화 폴더로 이동 |

---

## 파일 전달 흐름

```
메일 입사서류·인사 자료 수신 → 01_수신함 임시 보관 (당일 이동 필수)
→ 암호화 폴더 이동 → 포털 인사발령 결재 → NAS HR 전용 드라이브
```

---

## 폴더 구조

```
📁 HR                                      ← HR 담당자 전용 접근 권한
  📁 00_업무보고                           ← 최우선 노출
    📁 주간보고
    📁 월간보고
    📁 임원회의                            ← 개인식별 정보 포함 시 암호화
  📁 01_수신함                             ← 메일 첨부 임시 (당일 이동 필수)
  📁 20_채용
    📁 21_2025_공채
      📁 이력서                            ← 암호화 필수
      📁 면접자료
    📁 22_2025_경력
      📁 이력서                            ← 암호화 필수
      📁 면접자료
  📁 30_인사관리
    📁 31_입사서류
    📁 32_재직증명
    📁 33_퇴직
  📁 40_급여_연봉                          ← 팀장 이상 접근 권한만 부여
  📁 50_교육_훈련
  📁 90_Archive
    📁 91_3년보존
    📁 92_5년보존
```

## 파일명 규칙

| 유형 | 규칙 | 예시 |
|------|------|------|
| 이력서 | `지원자명_직군_YYMMDD` | `홍길동_백엔드개발_250601.pdf` |
| 입사서류 | `사번_성명_서류명` | `20250001_홍길동_재직증명서.pdf` |
| 인사발령 | `YYMMDD_발령내용_결재완료` | `250601_3팀발령_결재완료.pdf` |
| 보고서 | `YYMMDD_보고명_상태` | `250603_인력현황_확정.xlsx` |

## 운영 규칙

- **`01_수신함` 당일 정리** : 개인정보 파일이 수신함에 잔류하면 담당자 교체 시 접근 통제가 무너진다. 반드시 당일 업무 종료 전 해당 폴더로 이동한다.
- **개인정보 메신저 금지** : 이력서·연봉계약서·주민등록증 사본 등은 메신저 전달 금지. 반드시 사내 메일(암호화)로만 수발신한다.
- **급여·연봉 접근 제한** : `40_급여_연봉` 폴더는 NAS 접근 권한을 HR 팀장·임원으로만 제한하고, 일반 HR 담당자는 읽기 권한도 부여하지 않는 것을 권장한다.
- **임원회의 자료 암호화** : 개인식별 정보(인사 현황, 연봉 등)가 포함된 임원회의 자료는 암호화 후 `00_업무보고/임원회의`에 저장한다.
- **채용 이력서 보관 기간** : 채용 미확정자의 이력서는 채용 절차 종료 후 3년 보존 후 파기한다 (개인정보보호법 기준).
