# 구매·계약 — 업무 PC 파일 정리 가이드

> **적용 환경** : 폐쇄망 · 사내 업무 포털(그룹웨어) · 사내 메신저·메일 파일 첨부 전달 · NAS / SVN 공유 드라이브

---

## 핵심 원칙

| 원칙 | 내용 |
|------|------|
| 숫자 접두사 정렬 | `00_` → `01_` → `20_` 순으로 탐색기 정렬 고정 |
| 업무보고 최우선 | `00_업무보고`가 항상 최상단 노출 |
| 견적 당일 이동 | 메일 수신 견적서는 당일 PO 폴더로 이동 + 재명명 |
| 서명본·작업본 분리 | 최종 서명 계약서와 협의 중 수정본은 반드시 분리 |

---

## 파일 전달 흐름

```
메일 견적 수신 → 01_수신함 임시 보관 → 20_구매 PO 폴더 이동 + 재명명
→ 비교검토표 작성 → 포털 구매 품의 결재 → 계약 체결 → 90_Archive_완료
```

---

## 폴더 구조

```
📁 구매_계약
  📁 00_업무보고                           ← 최우선 노출
    📁 주간보고
    📁 월간보고
    📁 임원회의
  📁 01_수신함                             ← 메일 첨부 견적서 임시 보관
  📁 20_구매
    📁 PO-2025-031_서버증설
      📄 견적_A업체_250601.xlsx
      📄 견적_B업체_250601.xlsx
      📄 비교검토표.xlsx
      📄 품의서_결재완료.pdf
    📁 PO-2025-032_SW라이선스
      ...
  📁 30_계약서
    📁 31_원본_서명본                      ← 스캔 PDF (최종 서명본)
    📁 32_사본_작업본                      ← 협의 중 버전
  📁 90_Archive_완료                       ← 5년 보존 대상
```

---

## 파일명 규칙

| 유형 | 규칙 | 예시 |
|------|------|------|
| 견적서 | `견적_업체명_YYMMDD` | `견적_A업체_250601.xlsx` |
| 발주 품의서 | `품의서_PO번호_상태` | `품의서_PO2025031_결재완료.pdf` |
| 계약서 서명본 | `계약서_업체명_YYMMDD_서명완료` | `계약서_A업체_250620_서명완료.pdf` |
| 보고서 | `YYMMDD_보고명_상태` | `250603_구매현황_확정.xlsx` |

---

## 운영 규칙

- **`01_수신함`** : 메일로 견적서가 오면 임시 보관 후 당일 해당 `PO-번호` 폴더로 이동해 `견적_업체명_YYMMDD` 형식으로 재명명한다. 여러 업체 견적을 한 폴더에 모아야 비교검토표 작성이 쉽다.
- **서명본 분리 원칙** : 서명 완료 계약서는 `31_원본_서명본`에, 협의 중 수정본은 `32_사본_작업본`에 분리한다. 두 폴더를 섞으면 최종 서명본을 특정할 수 없게 된다.
- **품의서 보존** : 포털 전자결재 완료 후 품의서 PDF를 내려받아 해당 PO 폴더에 함께 저장한다.
- **5년 보존** : 계약 완료 건은 연말에 `90_Archive_완료`로 이동하고, NAS 별도 볼륨에 이중 백업해 법정 보존 5년(상법 기준)을 준수한다.
- **NAS 경로 공유 원칙** : 구매 현황 주간 보고는 `00_업무보고/주간보고`에 저장하고 메신저로 NAS 경로를 팀장에게 공유한다.
