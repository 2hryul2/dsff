# 임원 — 업무 PC 파일 정리 규칙

> **적용 환경** : 폐쇄망 · 사내 업무 포털(그룹웨어) · 사내 메신저·메일 파일 첨부 전달 · NAS / SVN 공유 드라이브

---

## 핵심 원칙

| 원칙 | 내용 |
|------|------|
| 숫자 접두사 정렬 | `00_` → `01_` → `20_` 순으로 탐색기 정렬 고정 |
| 업무보고 최우선 | `00_업무보고`가 항상 최상단 노출 — 모든 부서 보고 집중 |
| 상태 태그 파일명 | 수신 보고서에 `_검토중 → _검토완료` 태그로 진행 상태 명시 |
| 기밀 문서 분리 | 전략·기밀 문서는 접근 제한 폴더에 별도 보관 |

---

## 파일 전달 흐름

```
메일·메신저 보고 수신 → 01_수신함 임시 보관 → 00_업무보고 이동 + 상태 태그(_검토중)
→ 검토·지시사항 회신 → _검토완료 재명명 → 결재 완료본 90_Archive 이동
```

## 파일명 규칙

| 유형 | 규칙 | 예시 |
|------|------|------|
| 수신 보고서 (검토 중) | `YYMMDD_보고명_검토중` | `250603_주간보고_검토중.pptx` |
| 수신 보고서 (검토 완료) | `YYMMDD_보고명_검토완료` | `250603_주간보고_검토완료.pptx` |
| 월간 보고서 | `YYMM_보고명_확정` | `2505_월간경영보고_확정.pptx` |
| 임원회의 자료 | `YYMMDD_임원회의_최종` | `250520_임원회의_최종.pptx` |
| 외부기관 공문 | `YYMMDD_기관명_문서명` | `250601_금융감독원_조회요청.pdf` |
| 전략 문서 | `YYMMDD_프로젝트명_단계` | `250501_A사인수_검토안.hwp` |

## 운영 규칙

- **`01_수신함` 당일 정리** : 모든 부서에서 메일·메신저로 올라오는 보고 자료는 `01_수신함`에 우선 수신한 뒤, 당일 `00_업무보고` 하위 3개 폴더 중 해당 폴더로 이동하며 `_검토중` 태그를 붙인다.
- **상태 태그 관리** : 검토 완료 후 `_검토중` → `_검토완료`로 재명명한다. 비서가 있는 경우 이 명명 규칙을 비서와 반드시 공유한다.
- **기밀 문서 접근 제한** : `20_전략_기밀` 폴더는 임원 본인 계정만 접근 가능한 NAS 개인 드라이브를 사용하고 일반 공유 폴더와 분리한다.
- **외부 기관 공문 보존** : 법률·금융·감독기관으로부터 메일로 받은 공문은 수신 즉시 `30_외부기관` 하위 폴더에 이동하고, 원본 PDF 형태로 보관한다. 파일명에 발신 기관명과 수신일을 포함한다.
- **분기별 아카이브** : 분기 종료 시 처리 완료된 보고서와 결재 완료 문서를 `90_Archive`로 이동해 현재 업무 폴더를 가볍게 유지한다.
- **인사·조직 자료** : `40_인사_조직` 폴더는 HR 팀장 이상과 공유하되, NAS 접근 권한을 별도로 설정해 일반 구성원이 접근할 수 없도록 한다.
